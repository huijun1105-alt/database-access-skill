# database-access skill

This repository packages the `database-access` Codex skill as a standalone installable skill directory.

## Included files

- `SKILL.md`: Codex-compatible skill definition
- `SOURCE.md`: upstream source reference

## Upstream source

The skill content was adapted from:

`https://github.com/GPTomics/bioSkills/tree/ebe4f4003ebeef4beda84ae1fb4f52295848c4e1/database-access`

The upstream path contains a `README.md` but not a native Codex `SKILL.md`, so this repository wraps that content into Codex's expected skill format.

## Install

Copy this folder into your Codex skills directory, typically:

`~/.codex/skills/database-access`

Then restart Codex.
